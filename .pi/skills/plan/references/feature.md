## Feature Planning Lens

Use this reference when planning net-new user or system capability.

Check for:

- the primary user outcome and workflow change
- scope discipline for v1 versus later enhancements
- dependencies on existing interfaces, data, or platform behavior
- edge cases, degraded states, and rollout implications
- how success will be visible to users, operators, or reviewers

Prefer epics that deliver reviewable increments of user value rather than technical slices with no obvious outcome.
